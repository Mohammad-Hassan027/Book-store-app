## ✅ Requirements

## 🚩 Starting

```bash
# Clone this project
$ git clone https://github.com/Mohammad-Hassan027/Book-store-app.git

# Access
$ cd book-store-app

# Access
$ cd server

# Install dependencies
$ npm install

# Run the project
$ npm run start:dev

# The server will initialize in the <http://localhost:5000> 🚀

# Access
$ cd client

# Install dependencies
$ npm install

# Run the project
$ npm run dev

# The server will initialize in the <http://localhost:5173> 🚀
```
